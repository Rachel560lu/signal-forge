# Market Landscape: AI Opportunity Radar Tools

**Date:** 2026-05-09
**Question:** Does an open-source AI opportunity radar workspace already exist?

## Closest Existing Projects

| Project | What it does | Gap |
|---------|-------------|------|
| Trend Monitor (GitHub: dongzhang84/trend-monitor) | Auto-aggregates GitHub Trending, HN, Product Hunt into digests | No AI analysis, no opportunity extraction |
| Robopapers Pain Points (Haptic-AI) | AI extracts pain points from papers → structured markdown output | One-shot, not a continuous system |
| Reddit Pain Point Analyzer | AI categorizes Reddit pain points | Single source only |
| Trendshift.io | Multi-source repo trend ranking | Repo discovery only, no opportunity framing |

## What Doesn't Exist (The Gap)

No single open-source project combines:
- Multi-source signal collection (HN + Reddit + GitHub + ProductHunt)
- AI-assisted pain point extraction + categorization
- Structured markdown workspace with AI context (CLAUDE.md pattern)
- Continuous loop: collect → analyze → surface "build this" opportunities
